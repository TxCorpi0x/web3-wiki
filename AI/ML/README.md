# Machine Learning

This directory contains Machine Learning (ML) notes, patterns, and practical references collected for the project.

## Purpose

- Provide concise conceptual overviews for ML subtopics.
- Collect practical tuning guidance, algorithm summaries, and reproducible experiment templates.
- Surface links to more detailed subdirectories and implementation notes.

## Subcategories

- **Federated Learning** — distributed training where clients keep data locally; see [Federated Learning](FL/README.md).

## How to use

- Browse subfolders for topic-specific READMEs and deep dives.
- Use the experiment log templates inside each topic (for example, Federated Learning) when recording runs.

If you'd like other ML subtopics added (optimization, vision, NLP, probabilistic models), tell me which and I'll scaffold READMEs.
